# Learn ES6 Features

Projeto simples para estudar algumas features do ES6, ES7, ES8. 
Utilizando Webpack 4 e Babel para estudar import/exports de modules no JS

## Features

```bash
ES6, ES7, ES8
JS Modules import/export
Webpack 4
Babel
```

## Installation

```bash
yarn or npm install
```

## Server

```bash
yarn server
```

## Development

```bash
yarn dev
```

## Development watch

```bash
yarn watch
```

## Build

```bash
yarn build
```
